# Backup App

Backup App is a script that is designed to be scheduled to run off a task scheduler. The app iterates through a specified directory, then compresses and saves user specified folders to a network drive.



# Installation

This program is mostly self contained within its folder. Where-ever you decide to place the folder, the path must be manually adjusted within the launcher.bat script.



# Usage

- Manually add the folder names to the list "folders" within the script.
- Double click on launcher.bat or schedule a program to execute this file.
